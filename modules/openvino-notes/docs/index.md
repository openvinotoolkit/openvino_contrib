# Заметки OpenVINO

**OpenVINO Notes** — это инновационное Android-приложение для создания и управления заметками, работающее на основе встроенного искусственного интеллекта.

Главное отличие проекта от аналогов — использование фреймворка **OpenVINO** для запуска нейросетевых функций непосредственно на мобильном устройстве (On-Device AI), без отправки приватных данных пользователя на сторонние серверы.

---

### 🎯 Цель проекта

Создание легковесного, конфиденциального и эффективного мобильного приложения для заметок, которое предоставляет пользователю инструменты умной обработки текста (суммаризация, выделение главного, автотеги) в офлайн-режиме.

### 🌟 Ключевые особенности

* **Конфиденциальность:** Локальный запуск ИИ гарантирует, что ваши заметки не утекут в сеть.
* **Автономность:** Приложение полноценно обрабатывает текст даже при полном отсутствии интернета.
* **Облачная синхронизация:** Надежный бэкенд для синхронизации медиафайлов и текстового контента между устройствами, когда сеть появляется.
